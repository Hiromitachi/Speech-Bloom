Create a modern, premium, mobile-first speech therapy web application called "SpeechBloom" inspired by the visual design language of Headspace, Calm, and modern wellness apps. The application should feel warm, encouraging, playful, and therapeutic rather than clinical or hospital-like. Use a soft pastel yellow theme as the primary color with complementary pastel lavender, pastel blue, pastel green, and off-white backgrounds. Typography should use Poppins, DM Sans, or Nunito with generous spacing, rounded corners (24px+), large cards, soft shadows, smooth transitions, and friendly illustrations. The UI should look polished enough for a real healthcare startup.

COLOR SYSTEM:
- Primary: #F8D979 (Pastel Yellow)
- Background: #FFFDF7
- Secondary Background: #FFF6D8
- Lavender Accent: #8B7CF8
- Blue Accent: #A8D8FF
- Green Accent: #C7F1D5
- Text Primary: #4A4A4A
- Text Secondary: #7A7A7A

The application should include authentication, user dashboard, therapist dashboard, progress tracking, exercise sessions, streaks, achievements, and detailed session history.

THERAPY SESSION EXPERIENCE
-------------------------------------------------

The app should guide users one exercise at a time.

Each exercise page contains:
- Exercise name
- Friendly illustration or animation
- Step-by-step instructions
- Repetition counter
- Live timer
- Progress bar
- Previous / Next controls
- Voice guidance placeholder
- Completion confirmation

Use smooth animations between exercises.
